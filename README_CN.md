## 电脑配置

| 配件 | 规格 | 工作状态 |
| :------------: | :------: | :--------: |
| 主板 | Asus PRIME B360M-A | ✅ |
| 处理器 | Intel Core i5-8400 @ 2.80Ghz | ✅ |
| 固态硬盘 | Hikvision C2000Pro 512GB | ✅ |
| 核芯显卡 | Intel UHD Graphics 630 | ✅ |
| 独立显卡 | NVIDIA GeForce GTX 1060 5G | 🚫 |
| 声卡 | Realtek ALC 887 | ✅ |
| 有线网卡 | Realtek RTL8111H | ✅ |

## 工作的部分

- macOS 10.15.4
- CPU 睿频
- 核芯显卡（HDMI 输出）
- 有线网卡
- 音频 （Layout=11）
- USB

## 不工作的部分

- 独立显卡（已屏蔽）
